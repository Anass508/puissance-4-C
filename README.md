# 🔴 Puissance 4 - Langage C

Ceci est une implémentation complète du célèbre jeu de stratégie **Puissance 4**, développée entièrement en **C**. Ce projet met en avant la logique algorithmique et la manipulation de tableaux bidimensionnels.

## ✨ Fonctionnalités
* **Mode 2 joueurs** : Affrontement local sur la même machine.
* **Validation des coups** : Système anti-erreur (colonne pleine ou inexistante).
* **Moteur de victoire** : Détection automatique des alignements de 4 jetons (lignes, colonnes et diagonales).
* **Interface Console** : Rendu visuel propre et mis à jour à chaque tour.

## 🛠️ Compétences techniques démontrées
* Utilisation des **tableaux 2D** pour la grille de jeu.
* Implémentation de fonctions de **vérification algorithmique**.
* Gestion des entrées/sorties utilisateur avec `stdio.h`.
* Compilation via **GCC**.
